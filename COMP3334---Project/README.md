# secure online storage system

一个安全消息传递系统，提供端到端加密的消息传递功能，确保消息的安全性、完整性和不可篡改性。

## 项目概述
本项目实现了一个基于xxx信息传递系统

## 系统架构

系统采用Client-Server架构，并结合区块链技术：

1. **客户端 (Client)**: 负责用户界面、本地加密和与服务器的通信
2. **服务器 (Server)**: 处理用户认证、消息路由和存储消息哈希和元数据，确保消息不可篡改
3. **加密**: RSA

## 项目分工

项目分为客户端和服务器端两个主要部分：


### Serve
